# Learning Hybrid-Control Policies for In-Contact Manipulation

Project page for the paper: **"Learning Hybrid-Control Policies for High-Precision In-Contact Manipulation Under Uncertainty"** by Hunter Brown, Geoffrey Hollinger, and Stefan Lee (Oregon State University).

## Setup

This is a GitHub Pages site. To deploy:

1. Push this repo to GitHub
2. Go to **Settings → Pages**
3. Under **Source**, select **Deploy from a branch**
4. Set branch to `main` and folder to `/ (root)`
5. Save — the site will be live at `https://<username>.github.io/<repo-name>/`
